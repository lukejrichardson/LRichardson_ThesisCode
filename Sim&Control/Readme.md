# To run the standard simulation:
1. Run Scripts/initVars.m
2. Open and run Simulation/SimControl.slx

# To run the Monte Carlo Simulations
1. Run Scripts/initVars.m
2. Open and run UncAttControllerSim.mlx to simulate the attitude controllers only.
3. Open and run UncPosControllerSim.mlx to simulate the attitude and position controllers.
